Struktur eines Aufsatzes

Общая структура — всегда одинаковая

✅ Золоте правило: ВСЕГДА эти 3 части!

1
Einleitung — вступление (3–5 предложений)

Название текста, автор, тема. По формуле TATTE.

2
Hauptteil — основная часть (80% текста)

Анализ / аргументы / описаниеание. Цитаты из текста.

3
Schluss — вывод (3–4 предложения)

Итог, своё мнение (если разрешено), общий вывод.

Aufgabentypen — какая структура нужна?

Hauptteil schreiben

Анализ, аргументы, цитаты

Для Typ 4a: Stilmittel analysieren

Формула: Stilmittel + Zitat + Wirkung

Der Autor verwendet [Stilmittel] („[Zitat]", Z. [Zeile]), um [Wirkung] zu erzeugen / zu verdeutlichen.

Пример: Der Autor verwendet eine Metapher („Herz aus Stein", Z. 5), um die Gefühllosigkeit der Figur zu verdeutlichen.

Для Typ 4b: Argument aufbauen (PEEL)

P
Point — утвержгдение

„Ein wichtiger Vorteil ist, dass..."

E
Explain — объяснение

„Das bedeutet, dass..."

E
Example — пример

„Zum Beispiel..." / „Laut dem Text..."

L
Link — связь с темой

„Deshalb ist dieses Argument wichtig für..."

Schluss schreiben

Шаблоны заключения

⚠️ Не гделай этих ошибок!

  • НЕ начинай Schluss словом „Zusammenfassend" — слишком банально
  • НЕ вводи новые аргументы в Schluss
  • НЕ повторяй дословно вступление
  • В Typ 2 — НЕ пиши своё мнение!

Шаблоны для Schluss

Für Typ 4a (Analyse):

Insgesamt lässt sich sagen, dass der Text [Kernaussage]. Durch den gezielten Einsatz von [Stilmittel] gelingt es dem Autor, den Leser [Wirkung]. Das Thema [Thema] bleibt auch heute noch relevant.

Für Typ 4b (Argumentation):

Abschließend bin ich der Meinung, dass [Hauptthese]. Zwar gibt es auch [Gegenargument], doch überwiegen meiner Ansicht nach die [Vorteile/Gründe]. Daher sollte man [Appell / Forderung].

Für Typ 2 (Sachtext):

Der Text macht deutlich, dass [Kernaussage]. Die dargestellten Informationen zeigen, wie wichtig das Thema [Thema] ist.

Полезные слова для Schluss

Insgesamt Abschließend Zusammenfassend lässt sich sagen Alles in allem Ich komme zu dem Schluss, dass Meiner Meinung nach

Nützliche Phrasen

Готовые фразы для каждой ситуации

Meinung äußern (выразить думку)

  • Meiner Meinung / Ansicht nach...
  • Ich bin der Meinung / Überzeugung, dass...
  • Ich halte es für wichtig, dass...
  • Ich finde / denke / glaube, dass...
  • Nach meinem Dafürhalten...

Argumente einführen (ввести аргумент)

  • Ein wichtiges Argument ist, dass...
  • Erstens / Zweitens / Drittens...
  • Hinzu kommt, dass...
  • Außerdem / Darüber hinaus / Zudem...
  • Ein weiterer Punkt ist...

Textbelege (цитаты из текста)

  • Im Text heißt es: „..." (Z. X)
  • Der Autor schreibt: „..."
  • In Zeile X wird deutlich, dass...
  • Dies zeigt sich in Z. X: „..."
  • Laut dem Text / der Grafik...

Gegenargument einräumen (признать контраргумент)

  • Zwar..., aber... / Zwar..., jedoch...
  • Auf der anderen Seite...
  • Allerdings muss man bedenken, dass...
  • Natürlich gibt es auch Nachteile: ...
  • Manche Menschen sind der Meinung, dass...

💡 Фразы для Figurenanalyse

  • Die Figur wirkt... / scheint... zu sein
  • Ihr Verhalten zeigt, dass...
  • Am Anfang ist sie..., doch am Ende...
  • Dies wird deutlich, als sie sagt: „..."
  • Die Beziehung zwischen [A] und [B] ist geprägt von...
