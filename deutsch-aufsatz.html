💡 TATTE — общая формула Einleitung

Все три типа начинаются с TATTE:

БукваЧто написатьПример
TTitel«Der Fremde»
AAutorvon Albert Camus
TTextartist ein Roman / Zeitungsartikel
TThemahandelt von Einsamkeit
EErscheinungsdatumerschienen 1942

⏱️ Тайминг (примерно 150 минут)

  • 10–15 мин — прочитать все 3 задания, выбрать тип
  • 30–40 мин — Leseverstehen-Aufgaben
  • 20–30 мин — план + структура для Aufsatz
  • 50–60 мин — писать сам Aufsatz
  • 15–20 мин — перечитать, проверить ошибки

Universal-Phrasen

Фразы, которые работают для всех трёх типов

Эти фразы пригодятся в любом типе. Специфические шаблоны для каждого типа — на их отдельных страницах.

Textbelege (цитаты из текста)

  • Im Text heißt es: «...» (Z. X)
  • Der Autor schreibt: «...»
  • In Zeile X wird deutlich, dass...
  • Dies zeigt sich in Z. X: «...»
  • Laut dem Text / der Grafik...

Übergänge (переходы между абзацами)

  • Zunächst möchte ich... / Im ersten Teil...
  • Außerdem... / Darüber hinaus... / Zudem...
  • Ein weiterer Punkt ist...
  • Auf der anderen Seite...
  • Im Vergleich dazu... / Ähnlich wie... / Anders als...

Erklärungen (объяснения)

  • Das bedeutet, dass...
  • Damit will der Autor sagen, dass...
  • Mit anderen Worten...
  • Das heißt / das bedeutet konkret...

Schluss-Einleiter (для заключения)

Insgesamt Abschließend Zusammenfassend lässt sich sagen Alles in allem Ich komme zu dem Schluss, dass

⚠️ Чего избегать в любом типе

  • Не начинай Schluss словом «Zusammenfassend» — банально
  • Не повторяй дословно вступление
  • Не вводи новые аргументы в Schluss
  • В Typ 2 — никакого личного мнения
  • Не пиши слишком короткие предложения подряд — используй Konnektoren
