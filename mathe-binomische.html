Binomische Formeln

Alle drei binomischen Formeln mit Beispielen

1. Binomische Formel

(a + b)² = a² + 2ab + b²
Beispiele:
(x + 3)² = x² + 6x + 9
(2a + 5)² = 4a² + 20a + 25
(3x + 4y)² = 9x² + 24xy + 16y²

2. Binomische Formel

(a − b)² = a² − 2ab + b²
Beispiele:
(x − 4)² = x² − 8x + 16
(5a − 2)² = 25a² − 20a + 4
(2x − 3y)² = 4x² − 12xy + 9y²

3. Binomische Formel

(a + b)(a − b) = a² − b²
Beispiele:
(x + 7)(x − 7) = x² − 49
(3a + 2)(3a − 2) = 9a² − 4
(5x + y)(5x − y) = 25x² − y²

⚠️ Typische Fehler

Fehler, die JEDER macht!

❌ Fehler 1: Mittleren Term vergessen

(x + 3)² = x² + 9 ← FALSCH!

(x + 3)² = x² + 6x + 9 ✅

❌ Fehler 2: Vorzeichen bei 2. Formel

(x − 4)² = x² + 8x + 16 ← FALSCH!

(x − 4)² = x² − 8x + 16 ✅

❌ Fehler 3: Koeffizient nicht quadrieren

(2x + 5)² = 2x² + 20x + 25 ← FALSCH!

(2x + 5)² = 4x² + 20x + 25 ✅

❌ Fehler 4: 3. Formel mit Quadrat

(x + 3)(x − 3) = x² + 9 ← FALSCH!

(x + 3)(x − 3) = x² − 9 ✅
