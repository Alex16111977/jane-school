Binomische Formeln

Alle drei binomischen Formeln mit Beispielen

1. Binomische Formel

(a + b)² = a² + 2ab + b²
Beispiele:
(x + 3)² = x² + 6x + 9
(2a + 5)² = 4a² + 20a + 25
(3x + 4y)² = 9x² + 24xy + 16y²

2. Binomische Formel

(a − b)² = a² − 2ab + b²
Beispiele:
(x − 4)² = x² − 8x + 16
(5a − 2)² = 25a² − 20a + 4
(2x − 3y)² = 4x² − 12xy + 9y²

3. Binomische Formel

(a + b)(a − b) = a² − b²
Beispiele:
(x + 7)(x − 7) = x² − 49
(3a + 2)(3a − 2) = 9a² − 4
(5x + y)(5x − y) = 25x² − y²
