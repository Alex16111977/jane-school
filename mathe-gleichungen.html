Gleichungen & Terme

Gleichungen lösen, Terme vereinfachen, Klammern richtig auflösen

Was ist ein Term?

Ein Term ist ein mathematischer Ausdruck mit Zahlen, Variablen und Rechenzeichen.

Beispiele:
3x + 5 Term mit Variable x
2(a + 4) − a Term mit Klammer

Was ist eine Gleichung?

Eine Gleichung hat ein = Zeichen. Beide Seiten haben den gleichen Wert.

3x + 5 = 20
Ziel: x alleine auf einer Seite

Terme vereinfachen

Zusammenfassen

Gleiche Variablen addieren/subtrahieren:
5x + 3x = 8x
7a − 2a + 3 = 5a + 3
4x + 2y − x + 5y = 3x + 7y

Klammern auflösen

Plus vor der Klammer → Vorzeichen bleiben:
+(3x − 2) = 3x − 2
Minus vor der Klammer → Vorzeichen drehen:
−(3x − 2) = −3x + 2
Faktor vor der Klammer → Ausmultiplizieren:
4(2x + 3) = 8x + 12
