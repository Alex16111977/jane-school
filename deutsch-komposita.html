Komposita — сложные слова

Donaudampfschifffahrtsgesellschaft • Handtuch • Hausaufgabe — как читать и строить длинные слова

📐

Das Grundprinzip

В немецком можно соединять слова без пробелов — и это абсолютно нормально

💡 3 главных правила Komposita

1. Последнее слово = главное — оно определяет род и основное значение:

das Hand + das Tuch = das Handtuch (полотенце для рук)

2. Первое слово уточняет/модифицирует главное:

Haus + Aufgabe = Hausaufgabe (задание для дома)

3. Род = род последнего слова:

der Hund + die Hütte = die Hundehütte

📋 Схема: Bestimmungswort + Grundwort

Bestimmungswort (уточняет) + Grundwort (главное) = Kompositum Перевод
Hand + Tuch = das Handtuch полотенце для рук
Haus + Aufgabe = die Hausaufgabe домашнее задание
Schul + Tasche = die Schultasche школьная сумка
Fahr + Rad = das Fahrrad велосипед («езда + колесо»)
Kühl + Schrank = der Kühlschrank холодильник («охлаждение + шкаф»)
Zahn + Bürste = die Zahnbürste зубная щётка
Brief + Kasten = der Briefkasten почтовый ящик

⚠️ Как определить род? Всегда по последнему слову!

der Schrank + die Kühl… → der Kühlschrank ✅ (не die!)

die Tasche + die Schul… → die Schultasche ✅

das Buch + die Schul… → das Schulbuch ✅

Unbekannte Wörter aufteilen

Метод разбивки — как расшифровать любое незнакомое слово

📍 Алгоритм из 4 шагов

1. Найди главное слово (последнее существительное) → определи род

2. Разбей слово на части (ищи корни)

3. Переведи каждую часть отдельно

4. Сложи смысл логически

Fugenlaute — связующие звуки

Маленькие элементы между частями сложного слова для удобства произношения

💡 Что такое Fugenlaute?

Иногда между частями сложного слова вставляется небольшой звук (или буква), чтобы слово легче произносилось. Они не меняют значение — только произношение.

📋 Все Fugenlaute

Fugenlaut Примеры Разбивка
-s- (самый частый) Arbeitsplatz, Lieblingsfarbe, Geburtstag Arbeits+Platz, Lieblings+Farbe, Geburts+Tag
-es- Bundestag, Landeshauptstadt Bundes+Tag, Landes+Hauptstadt
-e- Hundehütte, Schweinefleisch Hunde+Hütte, Schweine+Fleisch
-en- Blumentopf, Straßenbahn, Herzenswunsch Blumen+Topf, Straßen+Bahn
-er- Kindergarten, Bücherregal, Hühnerei Kinder+Garten, Bücher+Regal
-n- Augenarzt, Ohrenarzt Augen+Arzt, Ohren+Arzt

⚠️ Без Fugenlaut — запомни исключения

Некоторые слова пишутся без Fugenlaut, даже если кажется, что он должен быть:

❌ Handestuch → ✅ Handtuch

❌ Hausetür → ✅ Haustür

❌ Jahrestag (в значении «день рождения») → ✅ Geburtstag

К сожалению, единого правила нет — Fugenlaute нужно запоминать со словами.
