Alle Formen auswendig lernen — Inf. / Präs. / Prät. / Partizip II

Wie lernt man Verben? Für jedes Verb lerne: Infinitiv → er-Form (Präsens) → ich-Form (Präteritum) → Partizip II + haben/sein — учи 4 формы каждого глагола!

gelb = Vokalwechsel im Präsens (смена гласной) grün = Präteritum (прошедшее) lila = Partizip II (причастие II) blau = hat / rot = ist (Hilfsverb)
Tipp: Starke Verben haben einen Vokalwechsel: fahren → fährt → fuhr → gefahren. Schwache Verben enden im Partizip auf -t: machen → gemacht. Mischverben kombinieren beides: bringen → brachte → gebracht.

📚 Partizip II bilden — Regelübersicht

Schwache Verben: ge- + Stamm + -t
machen → gemacht  |  kaufen → gekauft
Bei -t/-d-Stamm: ge + Stamm + -et   arbeiten → gearbeitet
Starke Verben: ge- + (Stammvokalwechsel) + -en
fahren → gefahren  |  schreiben → geschrieben
Verben auf -ieren: Partizip ohne ge-!
passieren → passiert  |  studieren → studiert
Trennbare Verben: ge- kommt zwischen Präfix und Stamm
einladen → eingeladen  |  ausgehen → ausgegangen
Untrennbare Verben (be-, ge-, er-, ver-, zer-, emp-, miss-, ent-): kein ge-
vergessen → vergessen  |  benutzen → benutzt
haben oder sein?
sein: Bewegung von A→B  (gehen, fahren, kommen)
sein: Zustandsänderung  (werden, wachsen, sterben)
haben: alle anderen Verben
