📚

Plural в немецком

5 окончаний + возможный Umlaut

🎯 Главное

  • В Plural артикль всегда «die» (Nom., Akk.)
  • В Dat. Plural существительное получает -n: mit den Kindern
  • Род слова в Plural не виден — но при изучении нужно учить вместе с Plural
  • Правил много, но есть закономерности по роду

Запиши слово сразу с Plural

Когда учишь новое существительное — записывай так: der Tisch, -e / das Buch, ̈-er / die Frau, -en.

Знак ¨ над тире = Umlaut в Plural.

🔢

5 типов окончаний

С примерами и характерными словами

1. -e (часто + Umlaut)

  • der Tisch → die Tische
  • der Stuhl → die Stühle (Umlaut)
  • der Hund → die Hunde
  • das Jahr → die Jahre

2. -en / -n (никогда не Umlaut)

  • die Frau → die Frauen
  • die Schule → die Schulen
  • die Lampe → die Lampen
  • der Student → die Studenten

3. -er (всегда + Umlaut, если возможен)

  • das Kind → die Kinder
  • das Buch → die Bücher
  • das Haus → die Häuser
  • der Mann → die Männer

4. — (без окончания, иногда + Umlaut)

  • der Lehrer → die Lehrer (без изменений)
  • das Mädchen → die Mädchen
  • der Vater → die Väter (Umlaut)
  • der Apfel → die Äpfel (Umlaut)

5. -s (заимствования, иностранные слова)

  • das Auto → die Autos
  • das Hotel → die Hotels
  • die Oma → die Omas
  • der Park → die Parks
🔍

Закономерности по роду

Как угадать Plural, если не знаешь точно

🔵 der (мужской)

  • Чаще всего: -e (часто + Umlaut) — der Tisch / die Tische
  • На -er, -el, -en: — или Umlaut — der Lehrer / die Lehrer
  • Слабые (Student, Junge): -en

🌸 die (женский)

  • ~90% всех женских: -en / -n — die Frau / die Frauen
  • Несколько слов с Umlaut + -e: die Hand / die Hände, die Stadt / die Städte
  • На -in: -nen — die Lehrerin / die Lehrerinnen

🌿 das (средний)

  • Многие односложные: -er + Umlaut — das Buch / die Bücher
  • На -chen, -lein: — — das Mädchen / die Mädchen
  • Заимствованные: -s — das Auto / die Autos
